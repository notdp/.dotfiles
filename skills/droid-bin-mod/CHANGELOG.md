# Droid Binary Modifier Changelog

## 2025-01-11 - v0.46.0

### 修改内容

| # | 修改项 | 原始 | 修改后 | 说明 |
|---|--------|------|--------|------|
| 1 | 截断条件 | `if(!H&&!Q)` | `if(!0\|\|!Q)` | 核心：永远返回原文 |
| 2 | 命令阈值 | `length>50` | `length>99` | 更长命令显示 |
| 3 | 输出预览 | `slice(0,4)` | `slice(0,99)` | 99行预览 |
| 4 | diff行数 | `LD=20` | `LD=99` | Edit diff 99行 |

### 备注

- 修改1使截断函数永远走早期返回，后续逻辑不执行
- 因此原来的"截断返回"和"截断参数"修改不再需要

### 备份

- `~/.local/bin/droid.backup.0.46.0`
