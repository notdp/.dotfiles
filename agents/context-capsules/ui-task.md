# UI Task Capsule

This prompt looks like UI, CSS, React/Vue/Svelte, page, screenshot, Figma, or visual work.

- Load `/fe-ui-design` for implementation and `/fe-ui-visual-iterate` when comparing visuals.
- Do not claim UI is done with tests only.
- Capture real screenshots after changes, including mobile and desktop when relevant.
- Check horizontal overflow, text overflow, interaction states, and design-token adherence.
- If a Figma link is involved, use Figma MCP when available or suggest connecting it.
